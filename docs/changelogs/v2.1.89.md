# Changelog

## buddy

1. 运行 `/buddy hatch`，生成一个伙伴（companion），可以看到伙伴卡片
3. 运行 `/buddy` 或 `/buddy card`，看到卡片（包括属性数值和精灵图） 
4. 在卡片上按 `Esc` 或 `Enter`，关闭卡片 
5. 运行 `/buddy pet` 喂食 
6. 运行 `/buddy mute` / `/buddy unmute`，在终端侧边栏隐藏/展示伙伴

## token-budget

开启 Token 预算功能，允许用户在消息中指定 token 消耗目标，Claude 会持续工作直到接近预算上限，适合需要深度思考或大量编码的复杂任务。

用法：
- 在消息开头或结尾加 `+500k`、`+2m` 等简写形式
- 或使用完整形式：`use 500k tokens`、`spend 2m tokens`
- k = `千`，m = `百万`，b = `十亿`

## terminal-capture

开启终端面板捕获功能，允许 Claude 读取内置终端面板（Alt+J）中的内容。

功能：
- 新增 `terminal_capture` 工具，可捕获终端面板中当前显示的文本内容
- 支持 `lines` 参数控制捕获的滚动历史行数（1–10000）
- 通过 tmux 会话实现持久化终端面板的内容读取
- 工具为只读且并发安全，已加入 YOLO 模式安全白名单
